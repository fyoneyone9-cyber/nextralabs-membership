# 🧠 NextraLabs 失敗学ナレッジベース (ERROR_LOG.md)

このファイルは、開発中に発生したエラーとその解決策を蓄積し、二度と同じミスを繰り返さないための「システムの脳」である。

## 📋 累積トラブル・解決リスト

### 1. 文字化け（エンコード失敗）
*   **事象**: `AiSidejob.tsx` 等で日本語が化け、構文エラーでビルドが止まる。
*   **原因**: PowerShellの `Out-File` がシステム環境（SJIS等）に依存し、UTF-8を出力できないため。
*   **恒久対策**: **PowerShellでのファイル書き出しを厳禁とする。** 日本語を含むファイルの編集・作成は必ず **Node.js (fs.writeFileSync)** または **エディタツール (`edit`)** を使用し、強制的に `utf8` を指定する。

### 2. インポート漏れ（Symbol Not Found）
*   **事象**: `Badge`, `Button`, `CheckCircle2` 等が定義されていないというビルドエラー。
*   **原因**: UI刷新時に新しい部品を追加したが、ファイル冒頭の `import` 文の更新を忘れたため。
*   **恒久対策**: ファイルを書き出す直前に、使用している全てのコンポーネントとアイコンをリストアップし、**インポート行を自動生成・一括置換するフロー**を徹底する。

### 3. Google API 404/400 エラー
*   **事象**: Gemini API へのリクエストが拒否される。
*   **原因**: モデル名の不一致（1.5-flash vs 1.5-flash-latest）、URL構造のミス、Base64のヘッダー混入。
*   **恒久対策**: **「直接Fetch ＋ v1エンドポイント ＋ Base64完全クリーンアップ」** の黄金テンプレートを標準化する。

### 4. Vercel デプロイ制限
*   **事象**: `api-deployments-free-per-day` エラーで更新が止まる。
*   **原因**: 細かい修正のたびにプッシュを行い、1日100回の制限を超えたため。
*   **恒久対策**: **「複数修正を1つのコミットにまとめる」** 運用と、手動の `Redeploy` を自動化する **`Deploy Hook`** の活用を徹底する。
