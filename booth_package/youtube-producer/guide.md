# AI YouTubeプロデューサー [Standalone MASTER] 導入ガイド

この度は『AI YouTubeプロデューサー Standalone MASTER v2.1』をご購入いただき、誠にありがとうございます。
本パッケージは、NextraLabsが開発した「YouTube制作特化型AI」を、あなたのPC上で独立して動作させるためのデジタルアセットです。

## ?? パッケージ内容
1. `index.html`: ツール本体（ブラウザで開いて使用します）
2. `guide.md`: 本ガイドファイル

## ?? 準備するもの
本ツールを動作させるには、以下のAPIキーが必要です。
- **Google Gemini API Key**: [Google AI Studio](https://aistudio.google.com/) で無料で取得可能です。モデルは `gemini-2.5-flash` 以上を推奨します。

## ?? 使用方法
1. 本フォルダ内の `index.html` を、Google Chromeなどのモダンブラウザで開きます。
2. 画面上部の「SYSTEM ACTIVATION」エリアに、取得したGemini APIキーを入力します。
3. 「STRATEGY PALETTE」から、動画の方向性（戦略）を選択します。
4. 「SOURCE DATA」に動画の企画案や、文字起こしテキストを貼り付けます。
5. 「Generate Script」をクリックすると、AIがバズるための構成・台本を瞬時に生成します。

## ?? メリット
- **月額費用ゼロ**: あなた自身のAPIキーを使用するため、NextraLabsへの月額支払いは不要です。
- **完全プライバシー**: データはサーバーを経由せず、あなたのブラウザとGoogle AIの間で直接通信されます。
- **プロの戦略**: NextraLabs独自のプロンプトエンジニアリングが組み込まれています。
